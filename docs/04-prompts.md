# AI 활용 프롬프트 기록 (`04-prompts.md`)

본 문서는 MVTI 프로젝트의 아키텍처 및 핵심 기능을 설계하고 코드를 구현하기 위해 AI(Antigravity)와 주고받은 실제 프롬프트 및 수정 요청 흐름의 요약 기록입니다.

---

## 1. 프로젝트 요구사항 및 데이터 구조 설계 (Phase 1)
* **목적**: React CRUD 평가 기준에 맞추어, 영화 데이터가 아닌 사용자의 영화 반응 데이터(`UserMovieReaction`)를 조작 대상으로 하는 요구사항과 데이터 구조를 구체화함.
* **주요 프롬프트**:
  > "React를 이용해 영화 취향 진단 및 맞춤형 추천 웹앱(MVTI)을 만들려고 해. 과제 요구사항에 맞는 CRUD 대상 엔터티(UserMovieReaction)를 정의해주고, 8개 장르별 12편씩 총 96편의 영화 풀을 호환하는 movies.json 스키마와 요구사항 정의표를 마크다운으로 작성해줘."

---

## 2. TMDB API 영화 수집 스크립트 작성 (Phase 2)
* **목적**: 실시간 브라우저 API 호출의 장애 리스크와 키 유출을 방지하기 위해 로컬 일회성 수집 Node.js 스크립트를 작성함.
* **주요 프롬프트**:
  > "Vite 프로젝트의 로컬 환경에서 구동되며, 로컬 .env의 VITE_TMDB_API_KEY를 읽어 TMDB API에서 96편의 영화 정보를 수집하여 src/data/movies.json을 자동 생성하는 fetchMovies.mjs 스크립트를 작성해줘. 만약 로컬에 API Key가 설정되어 있지 않다면 미리 내장된 영화의 가상 데이터를 활용해 movies.json을 에러 없이 폴백 모드로 생성하도록 Node.js fs/fetch 기반으로 짜줘."

---

## 3. LocalStorage 및 CRUD Custom Hooks 설계 (Phase 3)
* **목적**: React의 상태 불변성을 지키며 LocalStorage와 동기화하고 예외 상황(JSON 파싱 오류 복구 등)을 방어하는 핵심 훅 개발.
* **주요 프롬프트**:
  > "React에서 state와 LocalStorage를 실시간 동기화하되, 데이터가 깨지거나 JSON 파싱 예외가 발생할 때 화면이 멈추지 않고 빈 배열로 복구하는 useLocalStorage Custom Hook을 짜줘. 그리고 이 훅을 상속하여 UserMovieReaction 엔터티의 추가(Create), 조회(Read), 수정(Update), 삭제(Delete)를 수행하며, 중복 추가 차단과 필수값 유효성 검증을 포함하는 useReactions Custom Hook을 작성해줘. 상태 변경 시 불변성 원칙(map, filter 등 사용)을 지켜줘."

---

## 4. 메이저 기획 수정: 명작 불호 일치도 기반 추천 (Phase 4-A)
* **목적**: 넷플릭스 등 기존 알고리즘의 한계를 극복하기 위해, 대중적 명작 중 본인만 독특하게 불호하는 코드가 일치하는 유저들을 대조해 추천하는 흐름 도입.
* **주요 프롬프트**:
  > "핵심 기획을 변경하고 싶어. 단순한 영화 호불호 취향 분석을 넘어, '남들은 다 명작이라 극찬하는데 나는 굳이 싫은 영화'가 일치하는 사람의 인생 영화 추천을 신뢰도가 높은 대안으로 유저에게 제안하려 해. 이 아이디어를 UI에 녹이기 위해, 영화 선택을 1단계(좋아하는 명작 선택 - 시안 테마)와 2단계(남들은 명작이라 하나 나는 굳이 싫은 영화 선택 - 주황색 테마)의 두 단계로 분리하여 진행하는 App.jsx와 SelectionTray.jsx를 재설계하고 연동하는 리액트 코드를 작성해줘."

---

## 5. 약식 MBTI 4문항 진단 탑재 (Phase 4-B)
* **목적**: 자신의 MBTI를 모르는 사용자를 위해 온보딩 과정 중 간단한 자가 진단을 수행할 수 있게 함.
* **주요 프롬프트**:
  > "내 MBTI를 모르는 유저들을 위해 E/I, S/N, T/F, J/P 성향을 1문항씩 총 4개 문항의 선택지로 판단하는 MbtiMiniTest.jsx 자가 진단 컴포넌트를 만들어줘. MbtiGridSelector.jsx에서 '약식 테스트로 알아보기' 버튼을 누르면 이 자가 진단 모듈이 렌더링되고, 답변이 끝나면 자동으로 조합된 MBTI 코드(ex: INFP)가 온보딩 선택값으로 전달되게 통합 연동해줘."
